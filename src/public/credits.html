Tools

Of course, I can’t thank Neocities enough for providing people a way to host their website!

I used their built-in editor at first, but soon switched to my preferred editor. It’s thanks to the Neocities CLI that I can edit this website on my computer and upload the files later.

Speaking of my preferred editor, I like Neovim! Unlike Neocities’ editor (as far as I know), it supports LSPs (language server protocols) that can spot errors in your code and provide suggestions. Neovim is based on Vim keybindings, so navigating text files with just the keyboard is quick and easy. There is a learning curve, but it’s fun once you get the hang of it. Plus, Neovim can be beautiful and very functional with the right plugins.

Finally, Browsersync helps me preview my website without pushing the changes to Neocities. There’s no worry that I’ll find my website broken only after I updated it. Not pushing files for every change is also less load on Neocities.

Design

I drew the background myself! It was inspired by the patterns here, which were made using a base by cairovercoat.

The site’s title icon and favicon are also by me, based on the background I made. Can you tell I love cats?

This website’s header font is Fira Code, and the body font is Karla.

The colors were modified from Catppuccin Macchiato. It’s a very nice pastel color palette!

The icons are from Font Awesome.

The overall design was inspired by TUIs (terminal user interfaces) and tiling window managers. I find them really charming for their simplicity and what they can do despite design constraints. There was not one single inspiration, but some of them include:

  • Hyprland, a dynamic window manager with great graphics
  • Dooit, a terminal to-do list
  • Arttime, a terminal time tracker with ascii art

They’re also just nice projects in their own right!
